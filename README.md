# Alumni-Management-Project
This project is an application made in Zoho Creator as no code/low code application

## Overview

This application enables colleges and universities to maintain a structured alumni database while simplifying communication, event management, and career opportunities. Using Zoho’s visual development environment, the application was created with minimal manual coding, allowing rapid deployment and easy customization.
